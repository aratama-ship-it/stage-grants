# 舞台芸術 助成金 適格性チェック（プロトタイプ）

舞台公演の団体・制作者が、自分の団体や企画に応募できる助成金を
「条件適合／要確認／対象外」の3段階で、根拠つきに判定できるWebアプリのMVPです。

## 公開URL
GitHub Pages で公開しています（`index.html` を開くだけで動作、外部依存なし）。

## これは何か
- 助成情報が乱立していて探しづらく、特に「自分がその条件に合うのか」の判断が
  舞台芸術の現場では大きなハードルになっています。それをカバーするための試作です。
- 単なる一覧・検索ではなく、**なぜ応募できる／できないのかを募集要項の条件ごとに示す**点、
  **締切と“いつ・どう入金されるか（支給時期）”を前面に出す**点が特徴です。

## 現在の収録範囲
- 検証済み **7制度**（全国＋東京の代表例。全39判定単位のうちの一部）
- データは **2026-07-17時点**で各公式サイト・募集要項PDFを一次確認したもの

## ご注意
- これは開発中のプロトタイプです。判定は募集要項の**明示条件のみ**に基づく機械的なもので、
  **採択可能性を示すものではありません**。
- 締切・金額・条件は変動します。最終的な応募判断は必ず各助成元の最新の募集要項でご確認ください。
